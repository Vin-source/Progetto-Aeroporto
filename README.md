# Progetto Basi di Dati e Object-Oriented – Aeroporto di Napoli

Questo progetto nasce come parte dell'esame universitario del corso di *Basi di Dati e Object-Oriented* (CdL Triennale in Informatica – A.A. 2024/2025). L’obiettivo è lo sviluppo di un sistema informativo completo per la gestione dell'aeroporto di Napoli.

L’applicazione è composta da:
- una base di dati relazionale
- un'applicazione Java con GUI (Swing)

Ogni Utente registrato all'applicazione può visualizzare i voli presenti, effettuare o modificare le proprie prenotazioni.
Ogni Amministratore del sistema può gestire i voli, crearli o modificarne i dati.
